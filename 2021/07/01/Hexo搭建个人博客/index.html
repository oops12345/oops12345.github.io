Hexo搭建个人博客

hexo 个人博客搭建

1. 安装必要组件

  • node.js
  • Git

2. GitHub端

  • 创建一个个人仓库 abc/abc/github.io (abc为GitHub个人用户名)
  • 到settings—git pages—choose theme，选择一个theme
  • 在浏览器输入abc.github.io,显示的应该是刚才选择的那个主题

3. 安装hexo本地文件

  • 选择电脑本地合适位置创建一个新的文件夹 来放置hexo的本地文件

  • 在该文件夹下右键选择 git bash here

  • 输入命令npm i hexo cli -g安装hexo文件

  • hexo -v 验证安装是否成功

  • hexo init初始化hexo本地文件

    tips: 错误console.js:39 throw new ERR_CONSOLE_WRITABLE_STREAM('stdout'); 由于node.js版本过低，去官网升级版本即可

  • hexo g生成本地静态页面

  • hexo s打开本地服务器，在localhost:4000查看结果

4. 连接Github与本地

  • git bash here 输入以下命令

    `git config --global user.name "oops12345"`

`git config --global user.email "oopsjava123@gmail.com"`

  • 生成ssh keys

    ssh-keygen -t rsa -C "oopsjava123@gmail.com"

  • 通过命令 cat ~/.ssh/id_rsa.pub输出密钥内容

  • 打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便。将上一步输出的密钥复制到这里

  • ssh -T git@github.com 检查链接是否正确

    error：git：The authenticity of host ‘github.com (13.250.177.223)’ can’t be established.

    在Are you sure you want to continue connecting (yes/no/[fingerprint])? 后输入yes 回车 即可解决

    参考地址

5. 更换主题及相关修改

  • 更换matery主题，参考官方Github仓库进行设置

  • 下载

    • 在博客根目录下themes文件夹下通过命令git clone https://github.com/blinkfox/hexo-theme-matery.git进行下载

    • 直接下载源码，解压、复制到theme文件夹下

  • 配置主题

    • 修改 Hexo 根目录下的 _config.yml 的 theme 的值：theme: hexo-theme-matery
    • 建议修改两个 per_page 的分页条数值为 6 的倍数，如：12、18 等，这样文章列表在各个屏幕下都能较好的显示。
    • 如果你是中文用户，则建议修改 language 的值为 zh-CN
